##### 安装brew https://brew.sh/index_zh-cn.html

##### 安装wget
```
$ brew install wget
```
安装nvm，安装node
##### 使用node
```
$ source ~/.nvm/nvm.sh
$ nvm use v7
```
##### 安装redis https://redis.io/download
```
$ wget http://download.redis.io/releases/redis-3.2.8.tar.gz
$ tar xzf redis-3.2.8.tar.gz
$ cd redis-3.2.8
$ make
```
##### 启动redes
```
$ src/redis-server
```
